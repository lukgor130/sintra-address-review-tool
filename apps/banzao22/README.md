# Banzao 22 AOI Review

Public URL: `https://maps.verrio.co/banzao22/`

Runtime folder: `/banzao22`

This route is a local-knowledge AOI review app generated from the Banzao 22 GeoJSON polygon supplied on 2026-05-19.
It uses the same V2 AOI viewer as Azenhas with a route-specific local pack at
`/banzao22/data/pack-banzao22`. The current pack contains 185 parcels with local street and satellite assets.
