# TerraVia Landing Page

Canonical home for the TerraVia land intelligence and design landing page.

Primary source folder: `/apps/terravia`
Target public URL: `https://terravia.verrio.co/`

The deployment bundle also exposes this app at `https://maps.verrio.co/terravia/` for workspace consistency, but the requested public URL is the dedicated TerraVia subdomain.

## UX Review Notes

- Added responsive layouts for mobile, tablet, and desktop.
- Replaced placeholder `#` links with section anchors so visible links lead somewhere predictable.
- Added accessible form labels, required fields, focus states, and reduced-motion support.
- Converted the contact form into a mailto handoff to `hello@terravia.pt` until a backend form endpoint is available.
